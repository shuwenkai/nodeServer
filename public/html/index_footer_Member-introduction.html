会员介绍

每一位会员都是不可缺少的一份子，成为我们的尊贵会员，可享受多项会员专属特权，优先获知新产品、促销和特别活动信息，定期获赠精美杂志等，尽享购物乐趣。

 

 

一、会员加入

 

1. 我想成为Tao尊贵会员；

您只需在官网在线注册或莅临门店填写“会员卡申请表”，即可成为Tao尊贵会员。

2. 我的会员卡如何积分；

持会员卡，您在门店或官网购物消费实付金额即可为您积分，不同商品积分不同(预付款及各类代金券等消费除外)。

3. 我的会员卡如何升级；

持会员卡，您在门店或官网购物实付消费金额(预付款及各类代金券除外)，会员消费累计达到相应金额即可自动升级，再次购物即可享受最新的会员折扣优惠：

客户实付消费金额累积达15 万，可升级为白金会员，再次购物即可享受白金卡折扣优惠；

客户实付消费金额累积达30 万，可升级为钻石卡会员，再次购物即可享受钻石卡折扣优惠；

会员购买商品退货或换货消费金额不满相应等级时，系统自动将会员卡调整至对应等级。

4. 我的会员卡积分如何查询。

您可以通过以下3 种方式查询：

( 1 ) 至门店服务台查询；

( 2 ) 致电全国服务热线查询；

( 3 ) 登录公司网站查询。

 

 

二、会员权利

走进门店消费您就是我们尊贵的客人，成为尊贵的会员将享受如下服务：


★ 普通会员

1.您的消费可累计积分，参加积分换赠活动；

2.凭卡可预约免费软装设计咨询服务，协助您完成整体家居设计和软装搭配；

3.可参加会员俱乐部举办的各项会员活动；

4.您将定期免费获得精美杂志；

5.定期获得各类新品、促销等活动信息。

★ 银卡会员

银卡会员除可享受普卡会员的全部权益外，还能享受以下优惠：

1.可享受所购正价商品9.8折优惠；

2.可享受所购商品现有促销折扣(特价、陈列样等商品除外)基础上的会员折扣或会员价等多项优惠；

★ 金卡会员

金卡会员除可享受普卡会员的全部权益外，还能享受以下优惠：

1.可享受所购正价商品9.5折优惠；

2.可享受所购商品现有促销折扣(特价、陈列样等商品除外)基础上的会员折扣或会员价等多项优惠；

★ 白金卡会员

白金卡会员除可享受普卡会员的全部权益外，还能享受以下优惠：

1.可享受所购正价商品9.2折折优惠；

2.可享受所购商品现有促销折扣(特价、陈列样等商品除外)基础上的会员折扣或会员价等多项优惠；

3.可获得免费养护服务一次(限门店所在城市服务范围内)；

4.您在生日当月将收到精心准备的一份生日；

5.可在特定活动时享受多倍积分（以当期活动为准）。

★ 钻石卡会员

钻石会员除可享受普卡会员的全部权益外，还能享受以下优惠：

1.可享受所购正价商品9.0折折优惠；

2.可享受所购商品现有促销折扣(特价、陈列样等商品除外)基础上的会员折扣或会员价等多项优惠；

3.可获得免费养护服务一次(限门店所在城市服务范围内)；

4.您在生日当月将收到精心准备的一份生日礼物；

5.可在特定活动时享受多倍积分（以当期活动为准）。

 

 

三、会员义务

1、会员在办理会员卡时请您提供真实的个人资料,以保障您的会员权益;若个人资料变更、丢失时,请及时与Tao联系(拨打全国客服热线或咨询门店)更新或挂失,以免造成您的积分损失。

2、会员结账时请主动出示本卡，为避免您的会员卡积分被盗用，在使用会员卡积分进行消费及查询积分时，请配合出示您的会员卡以供查询。

3、会员购买商品退货或换货时如有差价，顾客需退回会员卡相应的积分及赠品，积分随退货流程自动扣除，如若赠品不能归还，则按赠品零售价在退货货款中扣除。
